# Homebrew Tap: UltraXReal

Homebrew cask for [UltraXReal](https://github.com/DannyDesert/XReal-Ultrawide) — a free, open-source macOS menu bar app that creates a virtual ultrawide display for XReal Air AR glasses.

## Install

```bash
brew tap DannyDesert/ultraxreal
brew install --cask ultraxreal
```

## Update

```bash
brew upgrade --cask ultraxreal
```

## Uninstall

```bash
brew uninstall --cask ultraxreal
```
